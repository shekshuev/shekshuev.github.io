### 1. –ü–æ–¥–∫–ª—é—á–µ–Ω–∏–µ –Ω–∞–ø—Ä—è–º—É—é —á–µ—Ä–µ–∑ `connect()`

–§—É–Ω–∫—Ü–∏—è `psycopg.connect()` –ø–æ–∑–≤–æ–ª—è–µ—Ç —É—Å—Ç–∞–Ω–æ–≤–∏—Ç—å **–æ–¥–Ω–æ —è–≤–Ω–æ–µ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ** —Å –±–∞–∑–æ–π –¥–∞–Ω–Ω—ã—Ö PostgreSQL.

### 3. –†–µ–∑—É–ª—å—Ç–∞—Ç—ã –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è SQL-–∑–∞–ø—Ä–æ—Å–æ–≤

–ü–æ—Å–ª–µ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è `cursor.execute(...)` —Ä–µ–∑—É–ª—å—Ç–∞—Ç –∑–∞–ø—Ä–æ—Å–∞ –¥–æ—Å—Ç—É–ø–µ–Ω —á–µ—Ä–µ–∑ –∫—É—Ä—Å–æ—Ä.
–¢–∏–ø–∏—á–Ω—ã–π –æ–±—ä–µ–∫—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç–∞ –≤–∫–ª—é—á–∞–µ—Ç:

- `cursor.fetchall()` ‚Äî –ø–æ–ª—É—á–∏—Ç—å –≤—Å–µ —Å—Ç—Ä–æ–∫–∏;

- `cursor.description` ‚Äî –º–µ—Ç–∞–∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è –æ –∫–æ–ª–æ–Ω–∫–∞—Ö (–∏–º–µ–Ω–∞, —Ç–∏–ø—ã –∏ —Ç.–¥.).

#### –ü—Ä–∏–º–µ—Ä:

```python
    cur.execute("SELECT 1 AS one, 2 AS two;")

```

–ï—Å–ª–∏ –Ω—É–∂–Ω–æ –ø–æ–ª—É—á–∞—Ç—å —Å—Ç—Ä–æ–∫–∏ –≤ –≤–∏–¥–µ —Å–ª–æ–≤–∞—Ä–µ–π (`dict`), –º–æ–∂–Ω–æ –∑–∞–¥–∞—Ç—å `row_factory`:

```python
with conn.cursor(row_factory=psycopg.rows.dict_row) as cur:
    cur.execute("SELECT id, name FROM users;")
    for row in cur.fetchall():
        print(row["id"], row["name"])
```
