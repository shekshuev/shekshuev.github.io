### Вариант 7

**1. Что произойдет, если в предложении ORDER BY указать номер столбца (например, `ORDER BY 2 DESC`), а не его явное название?**

а) Запрос завершится с ошибкой, так как стандарт SQL требует явного указания имен столбцов.

б) СУБД отсортирует результат по второму столбцу, указанному в предложении `SELECT`.

в) СУБД отсортирует результат по второму столбцу физической таблицы в базе данных, независимо от того, что написано в `SELECT`.

г) Строки будут отсортированы в случайном порядке.

**2. Какая агрегатная функция позволяет объединить несколько строковых (текстовых) значений из разных строк в одну длинную строку, разделенную запятыми?**

а) `CONCAT_AGG()`

б) `STRING_AGG()` (или `GROUP_CONCAT()` в зависимости от диалекта СУБД)

в) `SUM_TEXT()`

г) `TEXT_JOIN()`

**3. Если в оконной функции отсутствует предложение PARTITION BY, как СУБД определит границы «окна»?**

а) Окно будет ограничено одной текущей строкой.

б) Функция выдаст синтаксическую ошибку, так как `PARTITION BY` является обязательным параметром.

в) Окном будет считаться весь результирующий набор данных (вся таблица целиком).

г) Границы окна будут определяться исключительно предложением `ORDER BY`.
