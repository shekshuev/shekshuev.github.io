**3. В чем разница между оконными функциями RANK() и DENSE_RANK()?**

а) `RANK()` работает быстрее, а `DENSE_RANK()` требует больше оперативной памяти.

б) При совпадении сортируемых значений `RANK()` пропускает следующие порядковые номера (например, 1, 2, 2, 4), а `DENSE_RANK()` не оставляет "дыр" в нумерации (1, 2, 2, 3).

в) `RANK()` сортирует по возрастанию, а `DENSE_RANK()` — по убыванию.

г) Нет никакой разницы, они взаимозаменяемы.
