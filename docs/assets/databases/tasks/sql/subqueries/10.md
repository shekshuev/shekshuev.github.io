а) Да, оконные функции можно использовать в любом месте запроса.

б) Да, но только если оконная функция используется внутри агрегатной.

в) Нет, оконные функции вычисляются после выполнения предложений `GROUP BY` и `HAVING`, поэтому могут находиться только в `SELECT` или `ORDER BY`.

г) Нет, оконные функции не поддерживают работу с агрегированными данными.

**3. Что означает запись `ORDER BY 1, 2 DESC` в конце SQL-запроса?**

а) Запрос завершится ошибкой из-за некорректного синтаксиса.

б) Результат будет отсортирован по первому столбцу по возрастанию (по умолчанию ASC), а при совпадении значений — по второму столбцу по убыванию.

в) Результат будет отсортирован по первому и второму столбцам, оба по убыванию.
